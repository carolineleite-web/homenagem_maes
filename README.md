# 💐 Página Especial - Dia das Mães

Um simples projeto para fazer uma homenagem ao dia das mães. A ideia foi criar uma landing page estática, com um visual limpo e bonito para servir como uma homenagem visual.

O site conta com uma abertura charmosa, uma seção destacando as qualidades de carinho, amor e força, e um espaço final com um visual elegante para uma cartinha ou mensagem especial.

## 🛠️ O que foi usado?
* **HTML5** estrutural e sem complicações.

## 🚀 Como testar e rodar
Não precisa configurar nada ou instalar dependências. Como é um projeto totalmente estático, você só precisa:

1. Baixar ou clonar o arquivo `index.html`.
2. Dar dois cliques nele para abrir direto no seu navegador de preferência.
